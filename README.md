# User Experience Testing & Evaluation

This repository contains **hands-on UX testing and evaluation work** completed as part of the **Software Quality Assurance & Test Engineering** program at **Conestoga College**.

The focus of this repository is on **applied UX practices**, including heuristic evaluation, usability testing, accessibility analysis, and user-centered design — demonstrated through real-world case studies and practical exercises.

---

## What This Repository Includes

### Heuristic Evaluations
- UX evaluations using **Nielsen’s 10 Usability Heuristics**
- Identification of usability issues with **severity ratings**
- Actionable improvement recommendations
- Real-world applications such as streaming platforms, academic portals, and content-based systems

### Usability Testing
- Task-based usability testing scenarios
- Clearly defined test objectives, tasks, and success criteria
- Analysis of user behavior, errors, and completion rates
- Findings translated into UX improvement suggestions

### UI & UX Analysis
- Evaluation of navigation, information architecture, and interaction flows
- Application of common **UI design patterns** (cards, dashboards, search, pagination, ratings)
- Analysis across **web and mobile interfaces**

### Accessibility & Inclusive Design
- Accessibility evaluations aligned with **WCAG guidelines**
- Identification of usability barriers for users with visual, hearing, motor, and cognitive impairments
- Recommendations for inclusive and accessible user experiences

### Lean UX & Agile UX
- UX evaluation work aligned with **Agile and Lean UX principles**
- Emphasis on iterative improvement, fast feedback, and usability-driven decision making
